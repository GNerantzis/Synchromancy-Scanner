# Synchromancy Scanner

Desktop application for scanning stocks, cryptocurrencies and commodities using custom trend-following algorithms.

## Features

- Market scanning for stocks, crypto and commodities
- Trend detection using Supertrend-based logic
- Backtesting engine
- SQLite factor caching
- Multithreaded processing
- PySide6 graphical interface
- TradingView integration

## Technologies

- Python
- PySide6
- SQLite
- yfinance
- ccxt
